# Telegram RSS Bot

## 功能
订阅与退订RSS，更新的文章发送到Telegram

## 运行
根据`rss.sql`中的指令新建名为`rss.db`的数据库(若存在请先删除)。将token作为参数启动Bot即可。

[RSSBot](https://t.me/PythonRssBot)
